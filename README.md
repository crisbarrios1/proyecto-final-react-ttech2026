# Proyecto Final - React

## 📦 Import-tech

Tienda online de productos tecnológicos desarrollada con React.

---

## 🚀 Tecnologías utilizadas

- **React** (Vite)
- **React Router Dom** (navegación)
- **Firebase Firestore** (base de datos)
- **ImgBB** (alojamiento de imágenes)
- **CSS** (estilos personalizados)

---

## 📋 Funcionalidades

### 🔹 Productos
- Listado de productos desde Firebase
- Detalle de cada producto
- Imágenes alojadas en ImgBB

### 🛒 Carrito de compras
- Agregar productos al carrito
- Eliminar productos del carrito
- Ver total a pagar
- Finalizar compra

### 🔐 Panel de administración
- Acceso con contraseña: `admin123`
- Agregar nuevos productos
- Editar productos existentes
- Eliminar productos
- Subir imágenes para cada producto
